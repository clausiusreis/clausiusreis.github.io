Introdução ao HTML

O HTML, Linguagem de marcação para a World Wide Web, está em sua 5ª versão (HTML5)

Originalmente proposto em 1995 (Baseado em trabalho de 1993), vários grupos trabalharam em versões diferentes (XHTML 2.0, HTML 4.01) antes de se unirem no desenvolvimento do HTML5

HTML5 adicionou elementos para desvincular o uso de certos plugins, como o Flash, applets e players de vídeo

Linguagens de marcação

Definem formatos, maneiras de exibição e padrões dentro de um documento

Utilizam Tags para dar algum significado ao conteúdo a ser exibido

Outro exemplo de linguagem de marcação é o XML, que tem foco na estruturação de dados

Sintáxe básica HTML

<tag>...</tag>

<tag atributos>...</tag>
