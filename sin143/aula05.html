Introdução ao CSS

Cascading Style Sheets (CSS) é usado para aplicar estilos sobre páginas Web. Pode ser utilizado de três formas:

  • Interno às tags HTML
  • Dentro das tags <style>...</style>
  • Em arquivos externos.

Linguagens de estilos

Linguagens de estilo, ou de folhas de estilo, é um mecanismo de expressar de forma estruturada o estilo visual de uma página Web. Seu grande benefício é a separação entre o conteúdo e a formatação visual, permitindo que designers possam modificar a "identidade visual" das páginas sem o risco de alterar o conteúdo das páginas.
