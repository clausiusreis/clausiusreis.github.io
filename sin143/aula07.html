Display

A propriedade display é uma das mais importantes no CSS para o controle da exibição de layouts.

O valor padrão da maioria dos elementos é
block ou inline

Display

Elementos por padrão block:

div, h1-h6, p, form, header, footer, section


Elementos por padrão inline:

span, a, img

Position

A propriedade position especifica o método de posicionamento de um elemento.

Os valores podem ser:

  • static
  • relative
  • fixed
  • absolute
  • sticky
