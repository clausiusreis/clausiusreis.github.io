What is PHP?

  • PHP é um acrônimo para "PHP: Hypertext Preprocessor"
  • PHP é uma linguagem de script open-source amplamente utilizada
  • Cógidos PHP são executados no lado do servidor
  • PHP é grátis para download e uso
