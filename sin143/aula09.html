O que é Javascript?

Linguagem de programação que torna possível implementar comportamentos complexos em páginas web. É capaz de tornar conteúdos atualizáveis em intervalos de tempo, mapas interativos, gráficos 2D e 3D animados, entre muitas outras aplicações.

É uma dos três pilares do desenvolvimento web, sendo peça fundamental de tecnologias atuais que se tornaram padrão o mercado.

O que o Javascript pode fazer?

  • Alterar conteúdo HTML de elementos
  • Alterar valores de atributos
  • Alterar estilos HTML
  • Esconde/Exibir elementos HTML
  • Manipulação de dados
  • Lógica da página
  • Entre muitas outras possibilidades...

Onde inserir código Javascript?

Dentro das tags <script>...</script>

Em um arquivo externo com o atributo src em <script>
<script src="MyScript.js" >...</script>

 

Podem ser inseridas em qualquer lugar do código, mas é importante ficar atendo às boas práticas de programação.
